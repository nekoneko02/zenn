# 本文
## 上級 MuleSoft デベロッパー（MCD2）とは
詳細は、[Trailhead](https://trailhead.salesforce.com/ja/credentials/mulesoftdeveloperii)、[受験ガイド](https://trailhead.salesforce.com/ja/help?article=Salesforce-Certified-MuleSoft-Developer-II-Exam-Guide)を参照ください。

簡単な情報：
- Salesforce(MuleSoft)のデベロッパー資格のうち、上位の資格
- 別名は、MCD2(MuleSoft Certified Developer - Level 2)
    - ※MCD1もあり、その上位資格がMCD2
- 上位のデベロッパー資格なので、本番稼働を見据えた実装関連が出題される

## 現時点のサンプルコード
以下の2つを作成しています：
- RAMLをExchangeへPublishする
- サンプルコードをCICDでデプロイする

READMEに動かし方は記載しているので、それを参考にExchangeのPublishやCICDでデプロイを実践できると思います。

## 勉強方法
- RAMLについては、公式のGitHubを見て勉強しています：
    - https://github.com/raml-org/raml-spec/blob/master/versions/raml-10/raml-10.md/
- CICD周りは以下
    - [Maven公式ページ](https://maven.apache.org/index.html)
    - ↑pom.xml, settings.xmlなどなど、気になるページを読み漁る
    - [GitHub 及び GitHub Actions を使った自動デプロイメント](https://jp.mulesoft-labs.dev/codelabs/auto-deployment-with-github-and-github-actions/)
    - [Mule Maven Plugin](https://docs.mulesoft.com/jp/mule-runtime/latest/mmp-concept)

## 試験の感想
RAMLに関する知識は、そこまで試験で問われない印象です。
CICD周り(というよりMavenビルド)は、基本的なMavenの知識に加えてMule Maven PluginやMUnit Maven Pluginも知っておく必要があるようでした。
